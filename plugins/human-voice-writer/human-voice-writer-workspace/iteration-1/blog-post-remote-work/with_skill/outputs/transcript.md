# Work Process Transcript

## Task
Write a 3-paragraph blog post about why remote work is here to stay. Target audience: HR professionals.

## Step 1: Read the Skill and Reference Materials

Read the main skill file at `/tmp/human-voice-writer-skill/human-voice-writer/SKILL.md`, then read both reference files:
- `references/ai-tells.md` (banned vocabulary, banned structural patterns, statistical deep patterns organized by detection tier)
- `references/humanization-techniques.md` (technique library covering word, sentence, structure, and tone layers, plus the 16-dimension voice framework)

## Step 2: Determine Register and Format

- **Format:** Blog post, 3 paragraphs
- **Register:** Professional but conversational (HR audience reading a blog, not a white paper)
- **Length calibration:** 3 paragraphs as specified. Each paragraph should be substantive but not bloated.

## Step 3: Write First Draft Applying All Seven Core Rules

Applied the following while drafting:

1. **Ban the AI vocabulary:** Avoided all Tier 1 banned words (delve, leverage, foster, robust, utilize, nuanced, transformative, ensures, etc.). Used plain replacements throughout.
2. **Vary sentence length (burstiness):** Mixed short punches ("That's not speculation." at 3 words) with longer compound sentences (30+ words). Aimed for standard deviation of 8+ words per paragraph.
3. **Eliminate repetitive sentence patterns:** No anaphora chains, no vague "This" openers, no colon-to-tidy-list constructions, minimal semicolons.
4. **Break structural templates:** Paragraph 1 opens with a bold claim, Paragraph 2 opens with an observation about economic change, Paragraph 3 addresses the reader directly. No topic-sentence/elaboration/summary pattern repeated.
5. **Kill the assistant register:** Took a clear position (companies resisting remote work are losing). Used contractions (aren't, isn't, don't, hasn't). No hedging, no "both sides have merit."
6. **Eliminate deep grammatical AI signatures:** No trailing participial phrases. Active voice throughout. Minimal nominalizations.
7. **Calibrate length to format:** 3 paragraphs, appropriate for a blog post. No padding.
